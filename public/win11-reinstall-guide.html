Windows 11 全新安装教程

U盘安装 · 官方原版 · 零成本
电脑卡慢？中毒？换硬盘？彻底重装一台干净的系统

⏱ 下载 1~2h + 安装 30min 🔧 需 ≥8GB U盘 🆓 完全免费

1️⃣ 准备工作

💾一个 8GB 或以上 的 U盘（建议 USB 3.0，速度快很多）
⚠️备份好 U盘里的文件（制作启动盘会清空所有数据）
📁备份电脑上重要的文件（重装系统会清空 C 盘）
🌐品牌机（联想/华为/戴尔等）先去官网下载好网卡驱动备用
2

下载官方 Windows 11 ISO

微软官方渠道 — 正版无毒，完全免费

推荐方法：用微软官方下载工具（最简单）

  1. 打开浏览器，访问微软官方下载页面：
    https://www.microsoft.com/zh-cn/software-download/windows11
  2. 拉到页面，找到 "创建 Windows 11 安装媒体" 板块
  3. 点击 "立即下载"，下载 mediacreationtool.exe
  4. 运行工具 → 选语言和版本 → 选 "U盘" → 选你的 U盘 → 等待下载完成
✅ 工具自动从微软服务器下载最新官方 ISO — 正版、无毒、无捆绑

备选方法：手动下载 ISO
同样页面往下翻，找到 "下载 Windows 11 磁盘映像 (ISO)" → 下载 Win11_xxxx.iso（5~6GB）

3

制作启动 U盘

方式一（推荐）：mediacreationtool 一键完成，无需额外操作

方式二：如果你下载了 ISO 文件，用 Rufus 写盘

  1. 下载 Rufus（免费开源）：rufus.ie/zh_CN.html
  2. 插入 U盘 → 打开 Rufus
  3. 设备选择你的 U盘，引导类型 →"选择" → 找到 ISO 文件
  4. 其他保持默认 → 点击 "开始"
  5. 等待进度条走完（约 5-15 分钟）
💡 老电脑没有 TPM 2.0？Rufus 弹出选项时勾上 "绕过 TPM 检查"
4

设置 BIOS 从 U盘启动

把 U盘插到要重装的电脑上 → 重启 → 开机时疯狂按对应键
品牌启动菜单键BIOS 设置键
联想F12F1 / F2
华硕ESCF2 / Del
戴尔F12F2
惠普F9ESC / F10
华为F12F2
小米F12F2
神舟F7F2
组装机F11 / F12Del

在 Boot Menu 里选择你的 U盘（会显示品牌名或 USB HDD）→ 回车

✅ 屏幕变黑 → 出现 Windows 标志 → U盘启动成功！
5

安装 Windows 11

进入蓝色安装界面后，按步骤走
  1. 语言、时间、键盘 → 都选 简体中文 / 中国 → 下一步
  2. 点击 "现在安装"
  3. 激活 → 点 "我没有产品密钥"（装完再激活也可以）
  4. 选版本 → Windows 11 专业版 或 家庭版 → 下一步
  5. 勾选 "我接受许可条款" → 下一步
  6. ⚠️ 选 "自定义：仅安装 Windows（高级）"

⚠️ 分区操作

场景操作方法
全新安装 选中 C 盘分区 → 删除 → 小分区也一样 → 选 "未分配的空间" → 新建 → 下一步
保留 D 盘 只删除 ESP 分区（100-300MB）和 C 盘，不要动 D 盘
⚠️ 看不懂的分区不要乱删，搞错了数据就没了。

系统开始复制文件 → 安装 10~20 分钟 → 电脑会重启几次 → 不要动它

6

首次开机设置

电脑重启后进入 OOBE（首次设置向导）
  1. 选 中国 → 微软拼音 → 是
  2. 键盘布局 → 跳过
  3. 连接网络（如卡住：左下角点 "我没有 Internet 连接"）
  4. 给电脑命名
  5. 登录微软账户（想跳过：输入 test@test.com 会退回离线账户）
  6. 设置 PIN 码（登录密码）
  7. 隐私设置 → 全点 "否"
✅ 等待进入桌面 — 完成！

❓ 常见问题

Q：装完后要激活吗？
品牌机（之前自带正版 Win10/11）装完联网后自动激活。没激活的话：设置→系统→激活→疑难解答。同一台机器硬件授权绑定，不要再花钱买新密钥。
Q：装完没网怎么办？
网卡驱动没装上。用另一台电脑去品牌官网下载网卡驱动→U盘拷过来装；或用手机 USB 共享网络让 Windows 自动更新驱动。
Q：可以保留我的文件吗？
菜单有"保留个人文件"选项，但不建议选——保留会把旧问题也带过来。彻底清空重新来体验最好。
Q：装完后如何打驱动？
Win11 联网后自动更新大部分驱动。不放心的话：设备管理器→右键有问号的设备→更新驱动；或用品牌官方工具（联想电脑管家、华硕 MyASUS 等）。
