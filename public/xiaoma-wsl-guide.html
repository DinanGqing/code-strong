Windows 部署龙虾教程

小码（OpenClaw AI 助手）完整安装指南 — 专为小白设计，跟着走就行

⏱ 预计 20~30 分钟 📖 6 个步骤 🖥 Windows 10/11

📋 你需要准备什么

✅ Windows 10 或 11（64位）系统版本 2004 以上
✅ 空闲磁盘 10GB安装 WSL + Node + 小码
✅ 管理员权限装 WSL 需要，找电脑主人帮忙输一次密码

🧭 整套流程一览

Windows 电脑
    │
    ├── 第一步：PowerShell(管理员) ──→ wsl --install
    │                                        │
    │                                  [重启电脑]
    │                                        │
    ├── 第二步：打开 Ubuntu 终端 ──→ 设置用户名 + 密码
    │                                        │
    ├── 第三步：curl + sudo ──→ 安装 Node.js
    │                                        │
    ├── 第四步：npm + pnpm ──→ 安装包管理器
    │                                        │
    ├── 第五步：pnpm + openclaw ──→ 安装小码本体
    │                                        │
    └── 第六步：openclaw ──→ 🦞 小码上线！
1

📍 启用 WSL（Windows 里的 Linux）

WSL = 在 Windows 里面跑一个轻量 Linux 系统，小码就住在这个 Linux 里
  1. 按键盘 Win + R，弹出运行框
  2. 输入 powershell，然后按 Ctrl + Shift + 回车（以管理员身份运行）
    这时你会看到一个蓝色的窗口，标题是「管理员: Windows PowerShell」
  3. 复制下面这行命令，鼠标右键点击 PowerShell 窗口（会自动粘贴），然后回车
    powershell 
    wsl --install
  4. 看到如下画面就对了，等待下载完成（约 1~3 分钟）
    正在安装: 虚拟机平台
    已安装: 虚拟机平台
    正在安装: 适用于 Linux 的 Windows 子系统
    已安装: 适用于 Linux 的 Windows 子系统
    正在安装: Ubuntu
    请求的操作成功。直到重新启动前更改不会生效。
  5. 重启电脑（必须重启）
  6. 重启后，在开始菜单搜索 Ubuntu 并打开
    Installing, this may take a few minutes...
    Please create a default UNIX user account...
  7. 按提示操作：
    • 输入用户名：输入 xiaoma（或你喜欢的），回车
    • 输入密码：输完看不到字符是正常的，输完回车，再输一次确认
    ✅ 看到 xiaoma@DESKTOP:~$ 就成功了
💡 如果重启后没有自动弹出 Ubuntu，去开始菜单手动搜 Ubuntu 打开
2

📍 安装 Node.js（小码的发动机）

以下所有命令都在 Ubuntu 黑色终端 里执行，一行一行输入，每行输完按回车
  1. 先装 Node.js 的安装源：
    bash 
    curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
  2. 如果提示输入密码，就输你刚才设的那个密码，然后继续安装：
    bash 
    sudo apt-get install -y nodejs
  3. 检查安装成功：
    bash 
    node -v
    ✅ 看到 v22.xxx.x 就对了！（比如 v22.14.0）
3

📍 安装 pnpm（包管理器）

继续在 Ubuntu 终端里输入：
bash 
sudo npm install -g pnpm
pnpm -v
✅ 看到 10.x.x 或类似版本号就对了！
4

📍 安装 OpenClaw（这就是小码本人！）

⏳ 这一步可能要等 1~2 分钟，自动下载安装，耐心等
bash 
sudo pnpm install -g openclaw
openclaw --version
✅ 看到版本号就搞定！
5

📍 启动小码！

在 Ubuntu 终端输入：
bash 
openclaw

🚀 第一次启动会看到配置向导：

  • 会让你选择 哪个 AI 模型 → 推荐选 deepseek（便宜好用）
  • 需要你填 API Key → 看下面的说明

Agent: 小码 (main)

跟小码聊天：直接在终端里打字，回车发送
退出小码：按 Ctrl + C

6

📍 配一个网页聊天界面（可选）

终端聊天不够好看？配个 Web 页面：
bash 
mkdir ~/xiaoma-web && cd ~/xiaoma-web
pnpm create openclaw-app --template web-chat
npm run dev

然后打开浏览器，访问 http://localhost:3000，就能看到一个漂亮的聊天界面 🎉

❓ 常见问题

Q：安装 wsl 时说「需要启用虚拟化」？
去 BIOS 开启虚拟化技术：
重启 → 按 F2 / Del / F10（看开机画面提示）进入 BIOS
找到 Intel Virtualization Technology（Intel VT-x）或 SVM Mode（AMD）
设为 Enabled → F10 保存退出
Q：命令报错说「Permission denied」？
命令前面加 sudo 再试，比如 sudo apt-get install xxx
Q：apt-get update 很慢 / 下载超时？
这是网络问题，换个时间再试，或者让懂的人帮你配个国内镜像源
Q：wsl 装好了但是打不开 / 报错 WSL 2？
右键开始菜单 → 终端管理员 → 输入：
powershell 
wsl --update
然后重启
Q：启动 openclaw 后怎么跟小码聊天？
直接打字回车就行！小码会在终端里回答你
